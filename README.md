# digi-etch-a-sketch
a little drawing program i made using Python + Turtle in my free time. maybe will update when i feel like it.
## Features (for now)
- Colors (qwerty to change color)
- Thicknesses (1-4 to change thickness)
## Controls
- Backspace to clear
- Z,X for Pen Up and Pen Down respectively
- (QWERTY) to change color
- (1-4) to change size
